# أوامر تشغيل المشروع (Exam System)

إليك خطوات تشغيل المشروع بالكامل:

## 1. إعداد قاعدة البيانات (مرة واحدة فقط)
تأكد من تشغيل خادم MySQL (مثلاً عبر XAMPP).
قم باستيراد ملف قاعدة البيانات الموجود في المسار التالي:
`Backend/Scripts/Database.sql`

يمكنك فعل ذلك عن طريق phpMyAdmin أو عبر الأمر التالي في التيرمنال:
```bash
mysql -u root -p < Backend/Scripts/Database.sql
```

## 2. تشغيل الواجهة الخلفية (Backend)
افتح نافذة تيرمنال جديدة ونفذ الأوامر التالية:

```bash
cd Backend
dotnet run --urls "http://localhost:5000"
```
ستبدأ الخدمة بالعمل على الرابط: http://localhost:5000

## 3. تشغيل الواجهة الأمامية (Frontend)
افتح نافذة تيرمنال **أخرى** ونفذ الأوامر التالية:

```bash
cd Frontend
python -m http.server 3000
```
أو إذا كان لديك Node.js مثبتاً يمكنك استخدام:
```bash
npx serve -l 3000
```

## 4. فتح التطبيق
بعد تشغيل الخادمين، افتح المتصفح وتوجه إلى الرابط التالي:
[http://localhost:3000](http://localhost:3000)

---
**ملاحظات:**
- تأكد من بقاء نوافذ التيرمنال مفتوحة أثناء استخدام التطبيق.
- في حال واجهت مشاكل في الاتصال بقاعدة البيانات، تأكد من صحة البيانات في ملف `Backend/appsettings.json`.
